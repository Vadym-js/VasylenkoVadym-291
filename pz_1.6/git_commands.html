Create a Repository

From scrath -- Create a new local repository

$ git init [project name]

Download from an existing repository

$ git clone my_url

Observe your Repository

List new or modifield files not yet
committed

$ git status

Show the changes to staged

$ git diff

Show all staged and unstaged files

$ git diff --cached

Show all staged and unstaged
file changes

$ git diff HEAD

Show the changes between two commit: ids

$ git diff commit1 commit2

List the changes dates and authors
for a file

$ git blame [file]

Show the file changes for a commit
id and/for file

$ git show [commit]:[file]

Show full changes history

$ git log

Show change history for file inclucing dffs

$ git log -p [file/directory]

List all local branches

Merge branch_a into branch_b

Tag the current commit

$ git tag my_tag

Stages the file, ready for commit

Stages the file, ready for commit

$ git commit -m "commit massage"

$ git reset --hard
